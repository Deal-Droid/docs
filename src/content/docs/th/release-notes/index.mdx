ยินดีต้อนรับสู่บันทึกการอัพเดทของ DealDroid! คุณจะพบการอัพเดทล่าสุด ฟีเจอร์ใหม่ การปรับปรุง และการแก้ไขบั๊กสำหรับแพลตฟอร์ม AI Sales Agent ของเรา 🎉

ติดตามสิ่งใหม่ๆ และการปรับปรุงที่เราทำอย่างต่อเนื่องเพื่อประสบการณ์ที่ดีขึ้นของคุณกับ DealDroid

---

## 2026

### v1.1.6

**วันที่ปล่อย:** 9/มี.ค./2026

#### 🎉 ฟีเจอร์ใหม่

- **รองรับเวอร์ชันมือถือ:** เราได้เปิดตัวเวอร์ชันมือถือของ DealDroid ช่วยให้คุณจัดการความรู้ และ การโต้ตอบแชทได้ทุกที่ทุกเวลา ด้วยอินเทอร์เฟซที่ใช้งานง่ายและปรับแต่งให้เหมาะกับสมาร์ทโฟนและแท็บเล็ต

  ![Mobile Version](@/assets/release_notes/2026/mobile_version.png)

- **แสดงสัญลักษณ์ว่าบ๊อตต้องการความช่วยเหลือ:** เพิ่มสัญลักษณ์ (🤚) เพื่อแสดงว่าบ๊อตต้องการความช่วยเหลือ ช่วยปรับปรุงประสบการณ์ผู้ใช้และประสิทธิภาพการสนับสนุน

- **Pin Droid:** ผู้ใช้สามารถ pin droid ที่สำคัญไว้เพื่อเข้าถึงได้รวดเร็ว ช่วยเพิ่มประสิทธิภาพการทำงานและการจัดระเบียบ

- **รองรับ Whatsapp:** เพิ่มการรองรับการเชื่อมต่อ Whatsapp ช่วยให้ผู้ใช้สามารถสื่อสารกับลูกค้าผ่าน Whatsapp โดยตรงจาก DealDroid

#### ⚡ การปรับปรุง

- ปรับปรุงประสิทธิภาพและความเสถียรของการจดจำภาพ
- รองรับ Thirdparty API สำหรับลูกค้าภายนอก

#### 🐛 แก้ไขบั๊ก

- อีเมลไม่ได้บล็อกการส่งข้อความ

### v1.0.0

**วันที่ปล่อย:** 8/ก.พ./2026

#### 🎉 ฟีเจอร์ใหม่

- **Third-Party Async API**: เพิ่ม endpoint ใหม่เพื่อรับคำตอบจาก AI แบบ asynchronous ผ่าน webhook สำหรับการผสานรวมกับระบบที่มีขีดจำกัดเวลา timeout ได้ดีขึ้น

  ![การตั้งค่า Integration แสดงส่วน When AI Answer Ready](@/assets/third-party/integration_th.png)

- **AI Provider Redundancy**: ใช้ระบบ redundancy ที่เลือก AI provider ที่พร้อมใช้งานมากที่สุดในขณะนั้น เพื่อให้มั่นใจว่าการตอบสนองของ AI จะเสถียรและลดเวลารอของลูกค้า

- **ควบคุมพฤติกรรมบอตได้ง่าย:** เปิดตัวฟีเจอร์ที่ให้คุณกำหนดเงื่อนไขสำหรับการตอบสนองของ AI โดยไม่ต้องกำหนดเองใน intents หรือการตั้งค่าที่สำคัญ ช่วยให้การตั้งค่าง่ายขึ้นและเพิ่มความมั่นใจในการตอบกลับของ AI

#### ⚡ การปรับปรุง

- **Third-Party Webhook Debugging**: ปรับปรุงระบบ debugging สำหรับ webhooks ที่ส่ง

### v0.9.14

**วันที่ปล่อย:** 14/ม.ค./2026

#### 🎉 ฟีเจอร์ใหม่

- **Instant Answer**: ตอนนี้สามารถกำหนดได้ว่า ให้ทำงานเฉพาะเมื่อเป็นข้อความแรกของลูกค้าเท่านั้น

#### ⚡ การปรับปรุง

- **i18n**: แปลภาษา ไทย จีน ญี่ปุ่น สำหรับหน้า Edit ของทุกหน้า
  ![Translated edit page](@/assets/release_notes/2026/i18n.png)
- **Refactor**: ปรับปรุงภาษาที่ใช้ใน code ให้สะกดได้ถูกต้องทั้งหมด รวมถึงใส่ระบบตรวจภาษาอัตโนมัติในขั้นตอน CI/CD
- **LLM Testing**: ปรับปรุงระบบทดสอบ LLM เพื่อส่วนของการทดสอบอัตโนมัติเพื่อให้แก้ AI Code ได้เร็วขึ้น
- **AI Stubond**: ปรับปรุง AI ให้ตอบเฉพาะส่วนที่มีข้อมูลในระบบ ไม่ตอบทุกคำถามที่รู้

### v0.8.17

**วันที่ปล่อย:** 13/ธ.ค./2025

#### 🎉 ฟีเจอร์ใหม่

- **Chat Head**: สามารถเพิ่มช่องทาง Chat ผ่านทางเว็บไซต์ได้ และสามารถกำหนดสีและคำถามตั้งต้นได้ด้วย

- **Demo Chat Head**: เพิ่มลิงก์เพื่อแสดงหน้าตัวอย่าง Chat Head

#### ⚡ การปรับปรุง

- ป้องกัน AI ใช้ตัวอักษรซ้ำ เช่น ดีจ้าาาาาา Helloooooo
- มีระบบป้องกัน AI พูดข้อมูลถึงสินค้า และราคาที่เราไม่ได้ตั้งค่าไว้
- ปรับปรุงระบบการตรวจจับ intent ให้แม่นยำยิ่งขึ้น
- เพิ่มเติมให้ AI สามารถ UpSale สินค้าได้มากขึ้น
