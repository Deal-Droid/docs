ให้เปิดเว็บ line oa (manager.line.biz) พอ login เข้าไปแล้วก็ให้เลือก official account ที่เราต้องการจะเชื่อมต่อกับ dealdroid

![auto-response message, arrow point at Auto-response messages and point at Change at Chat Off](@/assets/getting_start/line/1_auto_response_messages.png)

จากนั้นกดที่ menu Auto-response messages แล้วกดที่ Change ที่ Chat Off เพื่อเข้าไป copy ค่าที่จำเป็นสำหรับการตั้งค่าใน dealdroid


![Response settings page, arrow point at Chat toggle button and point at Messaging API settings](@/assets/getting_start/line/2_response_settings.png)

ในหน้านี้ให้เราเปิด Chat toggle button ให้เป็น ON ก่อน แล้วกดที่ Messaging API settings เพื่อไปหน้าถัดไป
