ให้เปิดเว็บ line oa (manager.line.biz) พอ login เข้าไปแล้วก็ให้เลือก official account ที่เราต้องการจะเชื่อมต่อกับ dealdroid

![auto-response message, arrow point at Auto-response messages and point at Change at Chat Off](@/assets/getting_start/line_new/1_auto_response_messages.png)

จากนั้นกดที่ menu Auto-response messages แล้วกดที่ Change ที่ Chat Off เพื่อเข้าไป copy ค่าที่จำเป็นสำหรับการตั้งค่าใน dealdroid

![Response settings page, arrow point at Chat toggle button and point at Messaging API settings](@/assets/getting_start/line_new/2_response_settings.png)

ในหน้านี้ให้เราเปิด Chat toggle button ให้เป็น ON ก่อน แล้วกดที่ Messaging API settings เพื่อไปหน้าถัดไป

![Messaging API settings page, point at Enable Messaging API](@/assets/getting_start/line_new/3_messaging_api.png)

จะเห็นว่า Status ตอนนี้เป็น Disabled ให้เรากดที่ Enable Messaging API เพื่อเปิดใช้งาน

![Select provider popup, arrow point at Text Field and show name of provider](@/assets/getting_start/line_new/4_select_provider.png)

เมื่อกดปิด ระบบจะแสดงหน้าต่างให้เราเลือก Provider ก็ให้เราเลือก New provider แล้วใส่ชื่อ Line OA ของเรา จากนั้นกด Agree

![Privacy Policy and Term of Use popup, arrow point at Text Fields for URL input](@/assets/getting_start/line_new/5_privacy_policy.png)

แสดงหน้า Privacy Policy and Term of Use ในหน้านี้ให้เราใส่ URL ของ Privacy Policy และ Term of Use ของเรา (ถ้าไม่มีสามารถใส่ URL เว็บไซต์หลักของเราก่อนได้) จากนั้นกด Agree

![Enabling Messaging API popup, arrow point at OK button](@/assets/getting_start/line_new/6_enabling_message_api.png)

หน้านี้เป็นการยืนยันว่าเราจะเปิดใช้งาน Messaging API ให้เรากดที่ OK

![Dealdroid Channel settings page, arrow point at Setting icon, point at Channels menu, and point at New Line OA button](@/assets/getting_start/line_new/7_dealdroid_channels.png)

หน้า Dealdroid Channel ให้เรากดที่ Setting ของ Droid ที่มุมบนซ้ายครับ แล้วเลือกที่ Channels (ช่องทาง) แล้วกดที่ New Line OA เพื่อเริ่มการตั้งค่าเชื่อมต่อกับ LINE Official Account

![New Line OA setup page, arrow point at Channel Name field, Channel ID field, Channel Secret field, and Access Token field](@/assets/getting_start/line_new/8_dealdroid_line_setup.png)

พอเปิดหน้า New Line OA ขึ้นมาก็จะเห็นฟอร์มให้กรอกข้อมูลต่างๆ เริ่มจาก Channel Name, Channel ID, Channel Secret, และ Access Token โดยให้เราใส่ชื่อ Channel Name โดยแนะนำให้ใส่ชื่อเดียวกับ Line OA จะได้จำได้ง่าย ส่วน Channel ID และ Channel Secret ให้เราไปเอามาจากหน้า Messaging API settings ใน LINE Official Account Manager

![Messaging API settings page, arrow point at Channel ID and Channel Secret fields](@/assets/getting_start/line_new/9_dealdroid_channel_copy.png)

กลับมาที่หน้า Messaging API settings ใน LINE Official Account Manager เราจะเห็นข้อมูล Channel ID และ Channel Secret ให้เรากดที่ปุ่ม Copy ด้านข้างของแต่ละช่องเพื่อคัดลอกค่า

![Dealdroid Line OA setup page, arrow point at Channel ID and Channel Secret fields](@/assets/getting_start/line_new/10_dealdroid_channel_paste.png)

ให้เรานำค่าที่คัดลอกมาใส่ในฟอร์มของ DealDroid ในช่อง Channel ID และ Channel Secret ตามลำดับ

จากนั้นให้เรา copy webhook URL จากหน้า Dealdroid Line OA setup กลับไปที่หน้า Messaging API settings ใน LINE Official Account Manager

![Dealdroid Line OA setup page, arrow point at Webhook URL field](@/assets/getting_start/line_new/11_dealdroid_webhook_copy.png)

ให้เราเอา webhook URL ที่คัดลอกมาไปใส่ในช่อง Webhook URL ในหน้า Messaging API settings ของ LINE Official Account Manager แล้วกด Save

![Messaging API settings page, arrow point at Webhook URL field and point at Link to Line Developers Console](@/assets/getting_start/line_new/12_messageing_api_copy_webhook_line_dev.png)

จากนั้นเราอยากได้ Access Token เพื่อเอามาใส่ใน DealDroid โดยให้เรากดที่ Link Line Developers Consoles เพื่อไปยังหน้า Line Developers Console

![Line Developers Console page, arrow point at Provider and Channel](@/assets/getting_start/line_new/13_line_dev_console.png)

มาถึงหน้า Line Developers ให้เลือก Provider ที่เราได้สร้างไว้ตอนเปิดใช้งาน Messaging API จากนั้นเลือกที่ Channel ที่เราต้องการเชื่อมต่อ

//14_line_dev_messaging_api.png // Tab Messaging API Pointer ชี้ไปที่ tab Messaging API และชี้ลงล่าง เพื่อให้ scroll ลงไปดูปุ่ม Generate token

![Line Developers Messaging API page, arrow point at Generate token button](@/assets/getting_start/line_new/14_line_dev_messaging_api.png)

ในหน้า Channle ที่เราเลือกมาให้เลือกที่ tab Messaging API แล้ว scroll ลงมาด้านล่างจะเจอปุ่ม Generate token ให้เรากดปุ่มนี้เพื่อสร้าง Access Token ใหม่

![Line Developers Generate Access Token page, arrow point at Issue button](@/assets/getting_start/line_new/15_line_dev_issue_token.png)

พอลงมาด้านล่างแล้วจะเจอปุ่ม Issue ให้เรากดปุ่มนี้เพื่อสร้าง Access Token ใหม่

![Line Developers Access Token page, arrow point at Copy button next to Access Token field](@/assets/getting_start/line_new/16_line_dev_copy_token.png)

เมื่อสร้างเสร็จแล้วจะมี Access Token ขึ้นมาให้เรากดที่ปุ่ม Copy ด้านข้างของช่อง Access Token เพื่อคัดลอกค่า

![Dealdroid Line OA setup page, arrow point at Access Token field and Save button](@/assets/getting_start/line_new/17_dealdroid_copy_access_token.png)

กลับมาที่หน้า Dealdroid Line OA setup ให้เรานำค่า Access Token ที่คัดลอกมาใส่ในช่อง Access Token แล้วกด Save

![Line Official Account Response settings page, arrow point at Menu Response settings and Enable webhook toggle button](@/assets/getting_start/line_new/18_line_response_setting_enable_webhook.png)

ไปที่หน้า Line Official Account แล้วกดที่ Menu Response settings อีกครั้ง แล้วเลื่อนลงมาด้านล่างสุดเพื่อเปิดใช้งาน Webhook โดยกดที่ Enable webhook แล้วกด Save

![Line Official Account Auto-response messages page, arrow point at Home tab, Auto-response messages menu, and In use toggle button](@/assets/getting_start/line_new/19_line_disable_welcome.png)

ตอนนี้ระบบ Line ยังจะส่ง Message เริ่มต้นให้เรา อันนี้เราต้องไป Desable มันก่อน โดยไปที่หน้า Auto-response messages แล้วกดปิด ที่ช่อง In use ครับ

![Line application chat page of your Line OA showing message from user and reply from AI Agent](@/assets/getting_start/line_new/20_post_on_line.png)

ให้ทดลองต่อ Line เข้ากับ Line OA ของเรา แล้วลองส่งข้อความมาทดสอบดูครับ ถ้าทุกอย่างถูกต้อง AI Agent ใน DealDroid จะตอบกลับข้อความเราอัตโนมัติผ่านทาง LINE Official Account ของเรา

![Dealdroid Chat page showing message from user and reply from AI Agent, pointer at Chat menu](@/assets/getting_start/line_new/21_dealdroid_chat.png)

ถ้ามาดูที่ Chat ของเราจะมี Message เข้ามาที่หน้านี้แล้ว เป็นข้อความที่เราส่งไป และข้อความที่ AI Agent ตอบกลับมาอัตโนมัติผ่านทาง LINE Official Account ของเรา เท่านี้ก็เสร็จเรียบร้อยครับผม
