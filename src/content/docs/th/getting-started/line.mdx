พร้อมให้ AI Agent จัดการแชทลูกค้าบน LINE แล้วหรือยัง?

## ทำไมต้องเชื่อมต่อ LINE?

- 💬 **ตอบกลับทันที 24/7** — AI Agent ของคุณตอบข้อความลูกค้าแม้ทีมของคุณออฟไลน์
- ⚡ **ไม่พลาดโอกาสขาย** — การตอบกลับที่รวดเร็วช่วยเพิ่มการมีส่วนร่วมและโอกาสปิดการขาย
- 🎯 **อัตโนมัติการขายและตอบคำถาม** — AI สามารถแนะนำสินค้า ตอบ FAQs และนำผู้ใช้ไปสู่การซื้อ
- 📊 **รวมศูนย์การจัดการสนทนา** — ดูและจัดการการสนทนา LINE พร้อมช่องทางอื่น ๆ ทั้งหมดใน DealDroid

---

## 4 ขั้นตอนหลัก ของการเชื่อมต่อ LINE OA

- **Step A — เปิดใช้งาน [Messaging API & สร้าง Provider](https://manager.line.biz/)**: เปิด Messaging API ใน LINE OA และตั้งค่า Provider
- **Step B — สร้างช่องทางใน DealDroid**: เพิ่ม `New Line OA` ใน DealDroid และวาง `Channel ID` / `Channel Secret`
- **Step C — ตั้งค่า Webhook & Issue Token**: วาง `Webhook URL` ใน LINE, เปิด webhook และสร้าง Channel Access Token แล้ววางใน DealDroid
- **Step D — ทดสอบ & ปิด Auto-response**: ปิดข้อความต้อนรับของ LINE, ทดสอบจากหลายเบอร์, ตรวจสอบ Chat ใน DealDroid

---

## Step A — เปิดใช้งาน Messaging API & สร้าง Provider

1. เปิดหน้า Auto-response ใน LINE OA แล้วไปที่ `Response settings`

![auto-response message, arrow point at Auto-response messages and point at Change at Chat Off](@/assets/getting_start/line_new/1_auto_response_messages.png)

2. เปิด `Chat` toggle ให้เป็น ON แล้วคลิก **Messaging API settings**

![Response settings page, arrow point at Chat toggle button and point at Messaging API settings](@/assets/getting_start/line_new/2_response_settings.png)

3. หาก Messaging API ยังเป็น Disabled ให้กด **Enable Messaging API** และสร้าง/เลือก Provider

![Messaging API settings page, point at Enable Messaging API](@/assets/getting_start/line_new/3_messaging_api.png)

4. เลือก **New provider** (ถ้ายังไม่มี) ใส่ชื่อ Provider, กรอก URL ของ Privacy Policy/Terms แล้วยืนยัน

![Select provider popup, arrow point at Text Field and show name of provider](@/assets/getting_start/line_new/4_select_provider.png)

---

## Step B — สร้างช่องทางใน DealDroid และใส่ Channel ID / Secret

1. ใน DealDroid ไปที่ `Settings` → `Channels` → กด `New Line OA`

![Dealdroid Channel settings page, arrow point at Setting icon, point at Channels menu, and point at New Line OA button](@/assets/getting_start/line_new/7_dealdroid_channels.png)

2. เปิดฟอร์ม `New Line OA` เพื่อเตรียมใส่ `Channel ID`, `Channel Secret` และ `Access Token`

![New Line OA setup page, arrow point at Channel Name field, Channel ID field, Channel Secret field, and Access Token field](@/assets/getting_start/line_new/8_dealdroid_line.png)

3. กลับไปที่หน้า Messaging API ใน LINE แล้วคัดลอก **Channel ID** และ **Channel Secret** จากนั้นวางลงใน DealDroid

![Messaging API settings page, arrow point at Channel ID and Channel Secret fields](@/assets/getting_start/line_new/9_messageing_api_copy.png)

![Dealdroid Line OA setup page, arrow point at Channel ID and Channel Secret fields](@/assets/getting_start/line_new/10_dealdroid_channel_copy_webhook.png)

---

## Step C — ตั้งค่า Webhook และสร้าง Channel Access Token

1. ใน DealDroid คัดลอก `Webhook URL` ของช่องทางที่สร้างไว้ แล้ววางในช่อง `Webhook URL` ของ LINE Messaging API จากนั้นกด Save

![Dealdroid Line OA setup page, arrow point at Webhook URL field](@/assets/getting_start/line_new/11_messageing_api_copy_webhook.png)

![Messaging API settings page, arrow point at Webhook URL field and point at Link to Line Developers Console](@/assets/getting_start/line_new/12_messageing_api_copy_webhook_line_dev.png)

2. ไปที่ `LINE Developers Console` → เลือก Provider → เลือก Channel → ในแท็บ `Messaging API` ให้กด `Issue` เพื่อสร้าง Channel Access Token แล้วคัดลอก

![Line Developers Console page, arrow point at Provider and Channel](@/assets/getting_start/line_new/13_line_dev_console.png)

![Line Developers Messaging API page, arrow point at Generate token button](@/assets/getting_start/line_new/14_line_dev_messaging_api.png)

![Line Developers Generate Access Token page, arrow point at Issue button](@/assets/getting_start/line_new/15_line_dev_issue_token.png)

![Line Developers Access Token page, arrow point at Copy button next to Access Token field](@/assets/getting_start/line_new/16_line_dev_copy_token.png)

3. วาง `Access Token` ลงในช่อง `Access Token` ของ DealDroid และกด Save

![Dealdroid Line OA setup page, arrow point at Access Token field and Save button](@/assets/getting_start/line_new/17_dealdroid_copy_access_token.png)

---

## Step D — เปิด webhook, ปิด Auto-response และทดสอบ

1. ใน LINE OA → `Response settings` → เปิด `Enable webhook` แล้วกด Save

![Line Official Account Response settings page, arrow point at Menu Response settings and Enable webhook toggle button](@/assets/getting_start/line_new/18_line_response_setting_enable_webhook.png)

2. ปิด `Auto-response` (In use) เพื่อป้องกันข้อความต้อนรับอัตโนมัติรบกวนการทดสอบ

![Line Official Account Auto-response messages page, arrow point at Home tab, Auto-response messages menu, and In use toggle button](@/assets/getting_start/line_new/19_line_disable_welcome.png)

3. ทดสอบส่งข้อความไปที่ LINE OA ของคุณแล้วตรวจดูการตอบกลับจาก AI Agent ในหน้า Chat ของ DealDroid
![Line application chat page of your Line OA showing message from user and reply from AI Agent](@/assets/getting_start/line_new/20_post_on_line.png)

![Dealdroid Chat page showing message from user and reply from AI Agent, pointer at Chat menu](@/assets/getting_start/line_new/21_dealdroid_chat.png)

## Field mapping (ตารางอธิบายฟิลด์)

| ฟิลด์ (DealDroid) | หาได้จาก (LINE)                                              |
| ----------------- | ------------------------------------------------------------ |
| Channel Name      | ตั้งชื่อเองใน DealDroid                                      |
| Channel ID        | Messaging API → Basic settings                               |
| Channel Secret    | Messaging API → Channel secret                               |
| Webhook URL       | คัดลอกจาก DealDroid → วางใน LINE Messaging API               |
| Access Token      | LINE Developers Console → Messaging API → Issue (long-lived) |

<Aside type="tip">
  เก็บ `Channel Secret` และ `Access Token` อย่างปลอดภัย —
  ห้ามส่งผ่านช่องทางสาธารณะ
</Aside>

---

## Troubleshooting — ปัญหาพบบ่อยและการตรวจสอบ (ย่อ)

- Webhook ไม่ทำงาน: ตรวจสอบว่า `Enable webhook` เปิดและ `Webhook URL` เป็น https ถูกต้อง และ Save แล้ว
- AI ไม่ตอบ แต่ข้อความมาถึง DealDroid: ตรวจสอบ `Access Token` ว่าเป็น long-lived และตรวจดู logs ใน DealDroid Chat
- ข้อความตอบผิดพลาด: ตรวจสอบการแมปฐานความรู้และสคริปต์การขาย
- ไม่สามารถสร้าง Token/Secret: ตรวจสอบสิทธิ์บัญชีใน Provider
- Auto-response ส่งข้อความซ้ำ: ปิด `Auto-response` ใน LINE OA ก่อนทดสอบ

<Aside type="danger">
  หากพบปัญหาเกี่ยวกับ Token/Secret ให้รีเซ็ต Token ทันทีและอัปเดตค่าใน DealDroid
  เพื่อความปลอดภัย
</Aside>
