  </Card>
  <Card title="กำหนดหัวข้อต้องห้าม" icon="warning">

  </Card>
  <Card title="เชื่อมต่อกับแพลตฟอร์มข้อความ" icon="comment">

    Messenger](/th/getting-started/facebook) เพื่อเข้าถึงลูกค้าของคุณ
