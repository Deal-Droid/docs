## Step 3: Exchange Connection Credentials

Now we'll copy information back and forth between LINE Console and DealDroid. You'll need four pieces of data:

1. **Channel ID** â€” Your LINE OA's unique identifier
2. **Channel Secret** â€” A secret key for authentication
3. **Webhook URL** â€” The address where LINE sends messages to DealDroid
4. **Channel Access Token** â€” The key DealDroid uses to send messages back

---

### 3.1 Copy the Channel ID

![LINE Providers Dashboard](@/assets/getting_start/line/pic_4.png)

1. In the Console, select the **Providers** or **Dashboard** menu to view all your channels
2. Click on the **LINE Official Account** you want to connect

---

![LINE Channel Basic Settings](@/assets/getting_start/line/pic_5.png)

1. **Copy the Channel ID** displayed on the screen (it's a long numeric string)
2. **Paste it into the Channel ID field** in DealDroid

> **Note:** The Channel ID is located in the Basic settings section of your channel.

---

### 3.2 Copy the Channel Secret

![LINE Channel Secret](@/assets/getting_start/line/pic_6.png)

On the same page, scroll down slightly:

1. Find the **Channel secret** section and click the button to **Copy** the value
2. **Paste it into the Channel Secret field** in DealDroid

> **Important:** The Channel Secret is confidential. Never share it with anyone!

---

### 3.3 Configure the Webhook URL

This step is reversed â€” we copy from DealDroid and paste into LINE Console.

![LINE Messaging API Settings](@/assets/getting_start/line/pic_7.png)

1. **Copy the Webhook URL** from DealDroid
   > **Note:** Make sure the URL ends with your Channel ID number
2. Return to the LINE Console and click the **Messaging API** tab
3. Find the Webhook settings section and click **Edit** to open the popup
4. **Paste the Webhook URL** you copied from DealDroid
5. Click **Update** to save
6. **Enable Use webhook** by toggling it to Enabled (green)

> **Tip:** The Webhook URL is the address where LINE sends customer messages to DealDroid every time someone chats with your OA.

---

### 3.4 Copy the Channel Access Token

![LINE Channel Access Token](@/assets/getting_start/line/pic_8.png)

On the same **Messaging API** page, scroll to the bottom:

1. Find the **Channel access token (long-lived)** section
2. If no token exists yet, click **Issue** to generate one
3. **Copy the token** that appears (it will be a very long string)
4. **Paste it into the Channel Access Token field** in DealDroid

> **Important:** This token is critical â€” DealDroid uses it to send messages back to your customers. Keep it secure!
