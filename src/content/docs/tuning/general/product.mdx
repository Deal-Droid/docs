## How Your AI Uses Product Info

When you add a product, the information serves two main purposes:

1. **Keyword or Short Description:** The system first looks at all your product **Keywords** to get a quick overview. This helps the AI decide which products are most relevant to the customer's question at a glance.

2. **Full Description:** Once a product is chosen, the AI uses the **Full Description** and **Images** to provide specific, detailed answers to the customer.

This two-stage process ensures your AI can quickly narrow down options and then provide detailed, accurate information when needed.

## Product Fields Explained

Here's a field-by-field guide to adding a new product. Each field plays an important role in helping your AI understand and present your products effectively.

![Product listing page](@/assets/general/product_edit.jpg)

### Product Name

This is the official name of your product. Use the language your customers use, as the AI can handle translations on its own.

**Tips:**

- Include key details in the name if you have similar products
- Be specific and descriptive
- Use natural language that customers would search for

**Example:** `10kg Cold Pressed Dog Food ‚Äì Duck`

This is better than just "Dog Food" because it includes the size, type, and flavor ‚Äî all important details.

---

### Reference Code

This is an internal code (like a SKU) for your product.

**Example:** `G5BYV`, `ABCD-12`, `DOG-DUCK-10KG`

**Use cases:**

- Internal inventory tracking
- Integration with external APIs (like shopping cart systems)
- Order management and fulfillment

If you use an external API for things like managing a shopping cart, this code will be sent along for easy reference.

---

### Keyword or Short Description

This provides a brief description of the product to give the AI an overview of all products.

Think of this as a super-quick summary. What are the absolute key features? The AI uses this to get a fast overview when comparing multiple products.

**Example:**

```
10kg Cold Pressed Dog Food ‚Äì Duck, Easily Digestible, Hypoallergenic, Added Vegetables & Botanicals
```

**What to include:**

- Product name (if it contains important keywords)
- Key features and benefits
- Main differentiators
- Important specifications

**Keep it concise** ‚Äî aim for 1-2 lines that capture the essence of the product.

---

### Full Description

This is where you put all the juicy details about the product. Focus on the specifics of this single item.

**What to include:**

- Complete product specifications
- Detailed features and benefits
- Usage instructions or recommendations
- Materials, ingredients, or components
- Size, weight, dimensions
- Care instructions or important notes

**What NOT to include here:**

- General company information
- Shipping policies
- Return policies
- FAQs that apply to all products

> **Note:** If you have general information that applies to many products (like your company history or shipping policies), it's better to put that in the **Documents** section to keep things tidy.

**Example:**

```markdown
A complete nutrition cold pressed dog food, high in meat content with 'superfood' vegetables & botanicals. Our cold pressed method keeps natural enzymes and vitamins locked in as well as retaining fibre and molecular structures, and most important, dogs love the taste!
```

---

### Price

Simply enter the product's price as a number.

**Example:** `2000.00`

**Note:** The currency is configured in your main Droid Settings, so you don't need to include currency symbols here.

---

### Image & Cover Image

A picture is worth a thousand words! You can add two types of images:

- **Image:** A standard product photo showing the product clearly (e.g., package front, product details, ingredients label)
- **Cover Image:** A more promotional "hero" image that grabs attention (e.g., product in use, lifestyle shot, marketing banner)

**Image Guidelines:**

‚ö†Ô∏è **Keep it snappy!** To make sure your images load quickly for customers, try to keep the file size under 1MB. This avoids making the user wait.

**Best practices:**

- Use clear, high-resolution images
- Show the product from multiple angles if possible
- Include context (product in use, scale reference, etc.)
- Optimize file size without sacrificing quality
- Use JPG for photos, PNG for graphics with transparency

---

### Image Description & Cover Image Description

This text helps the AI (and visually impaired users) understand what's in the image.

**How DealDroid handles image descriptions:**

DealDroid automatically generates image descriptions for you using AI. When you upload an image and save:

- ‚ú® **If the description field is empty** ‚Äî DealDroid will automatically generate a detailed description based on what it sees in the image
- ‚úèÔ∏è **If you've written your own description** ‚Äî DealDroid will keep your text and won't overwrite it
- üîÑ **If you want DealDroid to regenerate** ‚Äî Simply delete the existing description (leave it blank) and click Save. DealDroid will create a new one for you

**Why image descriptions matter:**

- ü§ñ Provides more context to the AI for better product recommendations
- ‚ôø Improves accessibility for visually impaired users
- üéØ Helps the AI understand product features visible in the image
- üîç Better SEO if used on websites

**You can always edit the AI-generated description** to add specific details or match your brand voice.

**Best practices for writing image descriptions:**

- üìù **Start with the product name** ‚Äî This helps the AI identify which product the image belongs to
- üé® **Describe visual details** ‚Äî Colors, packaging, size, materials
- üì¶ **Mention key features shown** ‚Äî Labels, ingredients, benefits visible in the image
- üìè **Keep it concise but informative** ‚Äî 1-2 sentences is usually enough

**Example of a good image description:**

```
10kg Cold Pressed Dog Food ‚Äì Duck: A 10kg bag in purple packaging, showing duck as the main ingredient. The bag displays nutritional information and product benefits on the front.
```

**Notice how the product name comes first**, followed by descriptive details. This helps the AI understand and match the image to the correct product.

## Pro Tips for Success

### Use Markdown for Clarity

Structuring your **Full Description** with Markdown (using headers, bold text, and bullet points) helps the AI understand the information more easily.

**Markdown syntax:**

```markdown
# Main Header

## Subheader

**Bold Text**

_Italic Text_

To create a bullet list:

- Item 1
- Item 2
- Item 3

To create a numbered list:

1. First step
2. Second step
3. Third step
```

**Example formatted product description:**

```markdown
## Key Features

- **High meat content** with 'superfood' vegetables
- **Cold pressed method** preserves natural enzymes
- **Easily digestible** and hypoallergenic
- **Complete nutrition** for adult dogs

## Ingredients

Duck (60%), Sweet Potato, Vegetables (Spinach, Carrots),
Botanicals (Turmeric, Chamomile), Vitamins & Minerals

## Feeding Guide

| Dog Weight | Daily Amount |
| ---------- | ------------ |
| 5-10kg     | 100-150g     |
| 10-20kg    | 150-250g     |
| 20-30kg    | 250-350g     |
```
